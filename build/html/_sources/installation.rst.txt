Installation
============

1. **Clone the repository**:

   .. code-block:: bash

      git clone https://github.com/Elizabeth-Mwania/malaria_detection_with_hyperparamerter_tuning.git

2. **Install dependencies**:

   This project uses R. Install the required libraries as follows:

   .. code-block:: r

      install.packages(c("caret", "ROSE", "DMwR2", "ggplot2", "rpart", "rpart.plot"))

3. **Download the dataset**:

   Ensure the dataset file, `Malaria-Data.csv`, is available in the project directory.
